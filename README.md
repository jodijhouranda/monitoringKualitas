# Monitoring Kualitas BPS (Jodi Scraper)

Scraper otomatis untuk monitoring kualitas data survei BPS FASIH (Fasilitas Survei Terintegrasi).

**Fitur:**
1. Otomatisasi penarikan Datatable (Menu 2).
2. Otomatisasi penarikan Detail Data (Menu 3).
3. Penggabungan data ke dalam Template Excel.
4. Perhitungan kinerja PPL otomatis (Persentase Selesai Lapangan, Kecepatan Dokumen/Hari, dll).
5. Login otomatis tanpa perlu menekan tombol pause/resume berulang kali.
